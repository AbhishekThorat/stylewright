# Privacy Policy

_Last updated: 2026-06-16_

CSS Overrides is designed to collect as little as possible — in practice,
nothing.

## What we collect

**Nothing.** CSS Overrides does not collect, transmit, sell, or share any
personal data.

## What is stored, and where

- The CSS you write and the sites you apply it to are stored **locally on your
  device** using the browser's `chrome.storage.local` API.
- This data **never leaves your browser**. There is no server, no analytics, no
  telemetry, and no network requests of any kind made by the extension.
- You can export this data to a JSON file or import it yourself. That file is
  created and read only on your device, at your request.

## Permissions

- `activeTab` — lets the extension inject your CSS into the current tab, only
  when you click **Apply**.
- `scripting` — the API used to perform that injection.
- `storage` — stores your overrides locally, as described above.

The extension does not request access to read or change your data on all
websites. Access to a site is granted only when you act on that site.

## Removing your data

Uninstalling the extension removes all stored overrides. You can also clear
individual sites from within the panel.

## Contact

Questions about privacy? Open an issue or contact the maintainer.
